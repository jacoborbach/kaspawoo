# KaspaWoo

A beautiful landing page for the KaspaWoo WordPress plugin - the fastest, most secure way to accept Kaspa (KAS) cryptocurrency payments in WooCommerce.

## About

KaspaWoo is a free and open-source WooCommerce payment gateway plugin that enables online stores to accept Kaspa cryptocurrency payments. Built with security and ease of use in mind, it provides a seamless payment experience for both merchants and customers.

## Installation

This is a static landing page. To install the actual WordPress plugin:

1. Visit the [WordPress.org plugin page](https://wordpress.org/plugins/kaspa-payments-gateway-woocommerce/)
2. Click "Download" or install directly from your WordPress admin panel
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure your KPUB wallet address in WooCommerce ‚Üí Settings ‚Üí Payments ‚Üí Kaspa
